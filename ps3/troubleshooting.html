Troubleshooting

Quick PS3 browser and page-loading checks.

Exploit initialization failed

  1. Use the firmware-specific installer page when available.
  2. Set that exact installer page as the PS3 browser home page.
  3. Turn off “Confirm Browser Close”.
  4. Delete Cookies, Search History, Cache and Authentication Information.
  5. Close the browser completely.
  6. Open the browser again so the installer is the first page loaded.
  7. If it still fails, restart the PS3 and retry.

4.93: /installer/493alt.html

Page does not load

  1. Close extra browser windows.
  2. Clear browser cache and cookies.
  3. Open the exact PS3 mode URL again.
  4. Run the browser test page.

Buttons or layout look broken

Use the console page instead of desktop mode:

https://reprook.github.io/ps3-jailbreak/ps3.html

JavaScript test fails

Confirm JavaScript is enabled in the PS3 browser settings, reload the page and run the test again.
