USB Preparation

Basic storage and file-layout checks.

USB checklist

  1. Use a reliable USB storage device.
  2. Keep a separate backup of important personal files.
  3. Use the exact folder names required by the trusted guide you follow.
  4. Safely eject the drive from your computer before connecting it to the console.

Common problems

File not visible: Recheck the filename, folder path and storage format.

File appears damaged: Compare the checksum with the trusted release page.

USB not detected: Try another USB port or a different reliable drive.
