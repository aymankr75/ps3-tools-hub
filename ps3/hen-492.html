HEN 4.92 Installer

Preparation page before opening the supplied PS3 browser installer.

Important: Opening the installer page starts its process automatically. Use it only on a console showing firmware 4.92 with the matching HFW already installed.
PS3 FIRMWARE 4.92

Before you continue

  1. Open Settings, then System Settings, then System Information.
  2. Confirm the displayed firmware is exactly 4.92.
  3. Confirm the matching HFW was installed beforehand.
  4. Close other PS3 browser windows.
  5. Keep the console connected to stable power and internet.
  6. Run the browser test below before launching the installer.

When the process fails

  1. Close the PS3 browser completely.
  2. Clear browser cache and cookies.
  3. Restart the browser and return to this page.
  4. Do not repeatedly launch it if the firmware does not match.
