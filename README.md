# PS3 Flash Center v4

A dual-interface PS3 service hub for firmware 4.91, 4.92 and 4.93.

## Main links

- `index.html` — desktop/mobile interface
- `ps3.html` — lightweight PS3 browser interface
- `flash/4.93.html` — latest firmware route
- `flash/4.92.html` — supported route plus local legacy fallback
- `flash/4.91.html` — supported route
- `ps3/browser-test.html` — PS3 browser diagnostics

## Upload

Upload every extracted file and folder to the root of the existing GitHub repository.
Replace existing files when GitHub asks.

Do not upload the ZIP itself.

## Important

The launch pages require the user to confirm matching HFW before unlocking the maintained installer link.
