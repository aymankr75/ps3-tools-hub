Prepare HFW 4.93 with a USB

  1. On a PC, download the trusted matching HFW 4.93 update file.
  2. Format the USB drive as FAT32.
  3. Create this exact folder path: PS3/UPDATE/
  4. Place the update inside it with this exact name: PS3UPDAT.PUP
  5. Connect the USB to the PS3.
  6. Open Settings → System Update → Update via Storage Media.
  7. After installation, return here and select “HFW 4.93 is installed”.
